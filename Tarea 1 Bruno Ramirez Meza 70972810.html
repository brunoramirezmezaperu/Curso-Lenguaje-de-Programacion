Responder las siguientes preguntas

1.¿Qué es un Protocolos Http?

Es el protocolo de comunicación que permite las transferencias de información a través de archivos en la World Wide Web.

2.¿Qué es un Query String?

Es un término informático que se utiliza para hacer referencia a una interacción con una base de datos.

3.¿Qué es un Verbos HTTP?

Se utiliza para recuperar información desde un recurso.

4.¿Qué es un DNS?

Es un sistema de nomenclatura jerárquico descentralizado para dispositivos conectados a redes IP como Internet o una red privada.

5.¿Qué es un CDN?

Es una red de servidores que distribuye contenido de un servidor de "origen" a todo el mundo.
